ttc-scala
====

A command line tic-tac-toe game in scala.

### How to play

From the command line, type:
> sbt run

Sample game:

```text
     1   2   3
   -------------
1  |   |   |   |
   -------------
2  |   |   |   |
   -------------
3  |   |   |   |
   -------------
X's turn:
row>1
column>1
     1   2   3
   -------------
1  | X |   |   |
   -------------
2  |   |   |   |
   -------------
3  |   |   |   |
   -------------
O's turn:
row>
```
